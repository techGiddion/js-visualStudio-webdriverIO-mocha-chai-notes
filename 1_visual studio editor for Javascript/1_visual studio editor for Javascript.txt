***Node installation***

1. Open browser and type Download nodejs the below link should be visible [https://nodejs.org/en/download/]

2. Chose the node version>> go to windows based on operating system  >> install the Installer

3. Once the installation is done go add the node home path in system variable
	>>Edit system variables >> Advance >> new >> 
	Variable name: Node_Home
	Variable value : location of nodejs folder [typically C:\Program Files\nodejs\]
4. Might need to restart the system once, to verify NODEJS installed successfully go to CMD and type node -v [if you are getting the output with node version, its installed]
